# LegacyBridge Frontend Website

Updated static frontend package for LegacyBridge.

## Included Pages

- `index.html` — complete marketing website
- `privacy-policy.html` — privacy policy page
- `terms-and-conditions.html` — terms and conditions page mentioning Solution Planets
- `styles.css` — full responsive CSS
- `script.js` — mobile menu, FAQ accordion, section rendering, reveal animation
- `assets/legacybridge-logo.svg` — final logo lockup
- `assets/legacybridge-mark.svg` — app/icon mark

## Key Content Added

- Google Drive AppData storage positioning
- Trust/privacy pain messaging around third-party servers, spam and unsolicited calls
- 48-hour setup access pricing model
- Annual, lifetime and assisted setup pricing cards
- Expanded basic + technical FAQ
- Footer links to Privacy Policy and Terms & Conditions
- Product ownership mention: LegacyBridge is a product of Solution Planets

## CTA Routes

- `/login`
- `/claim`

## Deployment

This is a static website. Upload the full folder to your hosting/Vercel/Netlify/static server, or copy the files into your existing frontend project.
